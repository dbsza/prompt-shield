# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Prompt Shield, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/dbsza/prompt-shield/security/advisories/new) to submit your report. This ensures the issue can be triaged and addressed before public disclosure.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 7 days
- **Fix or mitigation:** as soon as possible, depending on severity

## Scope

This policy covers the Prompt Shield browser extension and its detection engine. It does not cover third-party dependencies — please report those to their respective maintainers.
